# Aftermath Test Task

A React + Vite frontend for Sui staking using the Aftermath SDK.

## Getting Started

```bash
pnpm install
pnpm start
```

## Build

```bash
pnpm build
```

## Stack

- Core: React 19, TypeScript, Vite
- Styling: Tailwind CSS v4, shadcn/ui
- SUI: Aftermath TS SDK, Mysten dApp Kit
- Server state and mutations: TanStack Query
